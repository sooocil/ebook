## ✨ Features

- 📖 View a list of books  
- ➕ Add new books  
- ✏️ Edit existing book details  
- ❌ Delete books  
- 🔍 Clean and simple UI

## 🛠 Tech Stack

- **Language:** Java  
- **IDE:** Android Studio  
- **Database:** SQLite (or Firebase if extended)
